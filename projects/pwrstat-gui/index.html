PwrStat GUI

PwrStat GUI is a Python application for viewing CyberPower UPS info on Debian-based operating systems. It uses CyberPower's pwrstat command to gather info about the UPS, and then displays that info for the user. It also displays the current load in a graph, and can log info for the user to access later. Version 1.0.0 was released in July 2025, and the project is licensed under the GNU General Public License v3.0. Downloading CyberPower's PowerPanel Personal for Linux is required to use this app. I am not affiliated with CyberPower or PowerPanel Personal.
